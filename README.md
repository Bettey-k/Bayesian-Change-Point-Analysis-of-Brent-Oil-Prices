# Bayesian Change Point Analysis of Brent Oil Prices

This project analyzes structural breaks in Brent oil prices using Bayesian change point detection.
It combines time series analysis with real-world geopolitical and economic events.

## Project Structure
- data/: raw and processed datasets
- scripts/: reusable analysis code
- notebooks/: exploratory and modeling notebooks
- reports/: written analysis and summaries
